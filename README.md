# Web Project Pokedex
## Aplicação web inspirada em uma Pokédex, desenvolvida com HTML, CSS e JavaScript
___________________________________________________________________________________
### Objetivo:
Desenvolver uma aplicação completa no front-end consumindo dados de uma API externa, aplicando:

- Estruturação com HTML
- Estilização com CSS
- Consumo de API com JavaScript

### Tecnologias utilizadas:

- HTML5
- CSS3
- JavaScript
- API REST (PokéAPI)
- Git e GitHub

### Estrutura e lógica do projeto:

- Este projeto simula uma Pokédex digital, onde os dados dos Pokémon são carregados dinamicamente a partir de uma API externa.
- Os dados recebidos são tratados e transformados em elementos visuais na página
- Cada Pokémon é exibido dinamicamente com base nas informações retornadas pela API
- A interface é construída com HTML e estilizada com CSS, criando uma experiência visual inspirada no universo Pokémon

### Estrutura do projeto:

/Web-Project-Pokedex-HTML-CSS-Js
├── index.html → estrutura da aplicação
├── style.css → layout e estilização
└── script.js → consumo da API e lógica da aplicação
